# Data Center Logistics Tablet

Developed by **Lzinsp**

A high-end logistics utility for the Data Center simulator. This mod introduces a portable terminal that streamlines hardware procurement and deployment.

## 🚀 Key Features
* **Point-to-Point Teleportation:** Advanced logic to deliver items directly to the technician's coordinate.
* **RGB Cable Synthesis:** Real-time color manipulation for CAT6E cables.
* **Smart Inventory:** Full integration with Servers, Switches, SFPs, and Racks.

## 🛠 Installation
1. Install [MelonLoader](https://melonwiki.xyz/).
2. Drop `DataCenterLogisticsTablet.dll` into your `Mods` folder.

## 📖 Usage
* Press **'T'** to toggle the tablet.
* Select hardware and it will spawn at your location.
